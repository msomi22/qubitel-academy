# Grade 1 CBC English — Writing Readiness Category & Quiz Bank

## Why this category is needed

The current Grade 1 English screen shows **Writing Readiness** as coming soon. This content should live under that learning area because the goal is no longer only reading words. The learner should now practise how letters and words are built.

Recommended learning path:

```txt
Alphabet Mastery
→ Reading Readiness
→ Writing Readiness
→ Language Use
```

## Proposed Learning Area

| Field | Value |
|---|---|
| Learning area title | Writing Readiness |
| Short description | Trace, copy, complete, and write simple letters and words. |
| Grade | Grade 1 |
| Subject | English |
| Status | Ready for implementation |
| Content type | Lessons + practice questions + exams |
| Main goal | Help learners connect sounds, letters, word parts, and early writing. |

## Suggested Writing Readiness Categories

| Category | Purpose | Suggested content |
|---|---|---|
| Letter Formation | Teach the child how each letter is written | A–Z uppercase and lowercase tracing |
| Word Tracing | Help the child trace simple words | bat, cat, dog, sun, pen |
| Copy Writing | Help the child copy words after seeing them | copy short CVC words |
| Missing Letter Practice | Help the child complete words | c _ t, _ a t, c a _ |
| Missing Word Part Practice | Help with advanced chunks | b _ _, g _ _ t, m _ _ n |
| Picture-to-Word Writing | Connect picture to spelling | 🐱 → cat |
| Dictation Writing | Hear word and complete/write it | hear “sun”, choose/write sun |
| Sentence Readiness | Copy and complete short sentences | I see a cat. |

## Implementation Rules

- The visible prompt must not expose the answer.
- The `readAloudText` may include the full word because the learner hears the word and completes the spelling.
- Every quiz must have a replay/speaker button.
- Every quiz must have four large mobile-friendly options.
- Missing-letter questions should support:
  - one missing middle letter: `c _ t`
  - one missing first letter: `_ a t`
  - one missing last letter: `c a _`
  - two-letter missing chunks: `b _ _`
  - two-letter middle chunks: `g _ _ t`
- Use existing Grade 1 read-aloud behavior.
- Use immediate feedback.
- Correct answer positions must be mixed.
- Keep the quiz screen compact enough for phone.
